# 🍲 Food Connect — Zero Waste, Full Hearts

**Food Connect** is a high-impact platform designed to bridge the gap between surplus food and those in need. By connecting donors (restaurants, caterers, households) with receivers (NGOs, community centers) and a dedicated network of volunteers, we aim to reduce food waste and feed hungry communities.

## ✨ Core Features
- **Role-Based Flexibility**: Sign in and choose your role (Donor, Receiver, or Volunteer) each session.
- **Real-Time Dashboards**:
  - **Donors**: Post surplus food with images and track listing status.
  - **Receivers**: Filter live food listings and request items with one click.
  - **Volunteers**: Accept requests, track delivery status, and mark items as delivered.
- **Comprehensive Admin Portal**: Monitor platform-wide stats, health, and manage distribution workflows.
- **Secure Authentication**: Power by Firebase Google Sign-in for users and a hardened Email/Password login for Administrators.
- **Dynamic Stats**: High-visibility impact tracking (Meals Shared, Active Volunteers, Live Listings).

## 🛠️ Technology Stack
- **Frontend**: React 19, Vite, Tailwind CSS (Vanilla CSS design system), Lucide React.
- **Backend/Auth**: Firebase (Authentication, Firestore, Storage).
- **Deployment**: Optimized for Vercel / Firebase Hosting.

## 🚀 Quick Deployment
1. **GitHub Sync**: Connect this repo to GitHub.
2. **Vercel Hook**: Import the repo into Vercel.
3. **Environment Variables**: Add your Firebase API keys to Vercel (or use the provided `vercel.json` for routing).
